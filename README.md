# 🚀 Paste App

A simple and powerful **React + Vite** based web application to create, manage, and share text pastes easily.

---

## ✨ Features

* 📝 Create new paste
* ✏️ Edit existing paste
* 🗑️ Delete paste
* 📋 Copy paste content
* 🔗 Share paste
* 🔍 Search pastes
* 📅 Date tracking

---

## 🛠️ Tech Stack

* ⚛️ React.js
* ⚡ Vite
* 🎯 Redux Toolkit
* 🎨 Tailwind CSS
* HTML
* CSS
* Java Script
